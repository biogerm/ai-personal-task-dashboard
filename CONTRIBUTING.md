# Contributing to Personal Dashboard

First off, thanks for taking the time to contribute! 🎉

## Development Setup

### Python Backend (Mac Sync & Flask)
1. Ensure you have Python 3.9+ installed.
2. Create a virtual environment: `python3 -m venv venv`
3. Activate it: `source venv/bin/activate`
4. Install dependencies: `pip install -r requirements.txt`
5. Run tests: `pytest`

### iOS Widgets (Scriptable)
1. Ensure you have Node.js and npm installed.
2. Install dependencies: `npm install`
3. Run tests: `npm test`

## Code Style

- **Python**: We follow [PEP 8](https://peps.python.org/pep-0008/). Please ensure your code is formatted properly and includes English docstrings.
- **JavaScript**: Use standard ES6 conventions. Keep code modular and testable.
- **Language**: All code comments, variables, and documentation MUST be in English.

## Submission Guidelines

- Fork the repository and create your branch from `main`.
- Write clear, descriptive commit messages.
- Ensure all tests pass before submitting a Pull Request.
- Update documentation (`docs/`) if you change features or architectures.
